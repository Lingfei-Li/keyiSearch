科亿搜索算法

输入：

  • 科亿的专利申请数据，每条数据拥有多个属性（例如：名称，申请人，状态）
  • 搜索关键词

输出：

  • 搜索结果，按照算法打分排序

算法过程(v1)：

            result = []
            split searchText into keywords by commas
            for patent in allPatents
                for key in patent.keys
                    val = patent[key]
                    for keyword in keywords
                        if val contains keyword
                            result.add(patent)
                            goto loopEnd;
                loopEnd:
                    break
            return result

优化：

            关键词 -> 近义词/替换词
            直接match、近义词match分数不同
            不同词语match的得分不同，比如:
                根据在数据中出现的次数打分
                记录被搜索的关键词。被搜索次数越多得分越高
            match多个词语加成
            match多个词语距离越近分数越高，最后按分数排序
            将match的部分高亮显示
